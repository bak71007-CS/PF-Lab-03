# Programming Fundamentals Lab

**Name:** Bilal Ahmed Khan 

**Roll Number:** 25K-6507

## Lab Objectives
1. Introduce students to GitHub and setting up their profile with their first repository.
2. Introduce the basic syntax and structure of the C programming language.
3. Teach students how to write and compile simple C programs.
4. Familiarize students with fundamental C programming concepts, such as variables and data types.
5. Develop problem-solving skills by writing C programs to solve basic computational problems.

## Tasks Completed
1. Created GitHub repository
2. Added and formatted README.md
3. Created a new branch of your roll no.
4. In your roll-number branch, create intro.c, add basic C structure, and commit with a
meaningful message.
5. Write a C program to print:
Welcome to Programming Fundamentals
Spring 2026
6. Task 6: Escape Sequences
Write a C program to print name, roll number, and section using \n and \t.
7. Task 7: Variables & Data Types
Write a C program declaring int, float, double, and char variables and print them using correct format specifiers.
8. Task 8: User Input
Write a C program that takes an integer, float, and character as input and displays them.
9. Task 9: Precision Control
Write a C program that takes a double input and prints it with default, 2 decimal, and 5 decimal precision.
10. Task 10: Pull Request & Merge
Create a pull request from your roll-number branch to main, merge it, and delete the branch.
